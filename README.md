# 🛡️ Ultimate AntiCheat

The next-generation lightweight AntiCheat for **Paper/Spigot 1.21+**  
Built for performance, precision, and fair gameplay.

---

## ✨ Overview

Ultimate AntiCheat is a modern, high-performance protection system designed to detect and prevent common Minecraft cheats without affecting server performance.

It uses a smart **violation-based system** to reduce false flags while keeping your server fully secure.

---

## ⚡ Features

- 🚫 Advanced cheat detection system  
- 🧠 Smart false-positive reduction  
- ⚡ Ultra lightweight & optimized performance  
- 📊 Real-time player behavior tracking  
- 🔧 Fully configurable checks & settings  
- 📢 Staff alert system (live notifications)  
- 🔒 Optional punishment system (kick/ban support)  
- 🎯 Designed for competitive servers  

---

## 🧩 Supported Checks

- Fly Detection  
- Speed Detection  
- KillAura Detection  
- Reach Detection  
- AutoClicker Detection  
- NoFall / Movement validation  

---

## 🎮 Commands

| Command | Description |
|--------|-------------|
| `/ac` | Main command |
| `/ac reload` | Reload configuration |
| `/ac alerts` | Toggle staff alerts |
| `/ac check <player>` | View player violations |

---

## 🔐 Permissions

| Permission | Description |
|------------|-------------|
| `ultimateac.admin` | Full plugin access |
| `ultimateac.alerts` | Receive cheat alerts |
| `ultimateac.bypass` | Bypass all checks |
